 # 🧮 Simple Math Calculator (Python)

A simple yet powerful **Python calculator** that performs essential mathematical operations including arithmetic, modulus, power, and trigonometric calculations like sine.

---

## 🚀 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧮 Modulus  
- 🔢 Power (Exponentiation)  
- 📐 Trigonometric Function: `sin(x)`  

---

## 🛠️ Technologies Used

- Python 3

---

## 📂 Project Overview

This project is a command-line based calculator built using Python. It helps beginners understand:

- Function usage  
- Conditional statements  
- User input handling  
- Working with Python's `math` module
